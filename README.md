# st - simple terminal

st is a simple terminal emulator for X which sucks less.

## Requirements

### Fonts:

* FiraCode Nerd Font Mono
* Monofur Nerd Font
* Symbola

### Tools:

* picom / compton (compositor)

## Patches

1. Xresources
0. Scrollback
0. Alpha
0. Font2
0. Bold != bright
0. Hide cursor
0. working dir
0. Anysize

## Custom Fixes

I changed some cursor color settings to make it easier to see regardless of the
color palette
